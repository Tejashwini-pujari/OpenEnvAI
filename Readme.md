# Customer Support RL Environment

## 📌 Description
This project simulates a real-world customer support system where an AI agent processes support tickets. The agent must classify the issue, determine urgency, and decide the correct action.

---

## 🎯 Tasks
- **Easy**: Clear and direct issues (e.g., payment failure)
- **Medium**: Slightly ambiguous technical issues
- **Hard**: Complex or multi-intent customer queries

---

## 🔍 Observation Space
The agent receives:

- `ticket` (string): A customer support message
